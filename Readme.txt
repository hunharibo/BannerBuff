BannerBuff - A Mount and Blade Bannerlord mod to add a custom resource and buff system to the game. 
It is possible to create custom resources other than what the game already contains (e.g. denars and influence).
Only Heros with the occupation of Lord (or Lady) may have custom resources.
The example implementation adds a resource type for casting spells - named magicka (but could be anything, spell points, mana, whatever).
